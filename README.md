# CS3219 Project (PeerPrep) - AY2526S1

## PeerPrep Overview

PeerPrep is a collaborative coding platform that allows users to prepare for technical interviews with their peers. It matches users in real-time based on problem difficulyt and topics preferences and enables users to solve code together with an integrated editor and real time chat.
